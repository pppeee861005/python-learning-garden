## 🚀 快速開始

### 📋 環境需求
- **Python 3.10+** 🐍
- **VSCode** 或其他程式編輯器 💻
- **Pygame** 遊戲開發庫 🎮

### 🛠️ 安裝步驟

1. **安裝 Python**
   ```bash
   # 前往 https://www.python.org/ 下載最新版本
   # ✅ 記得勾選「Add Python to PATH」
   ```

2. **安裝 Pygame**
   ```bash
   pip install pygame
   ```

3. **克隆專案**
   ```bash
   git clone [你的專案網址]
   cd 農場遊戲設計專案
   ```

4. **開始學習**
   ```bash
   # 從第一天開始
   cd day1-examples
   python hello_python.py
   ```
